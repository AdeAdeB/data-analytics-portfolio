# Austin Bike Sharing Data Analysis

## Objective
Analyze bike sharing usage patterns to identify peak periods, popular stations, and user behavior trends.

## Dataset
Austin bike-sharing dataset (CapMetro).

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Key Insights
- Peak usage occurs during weekday rush hours
- Subscribers account for the majority of trips
- Some stations are consistently underused
